Educational UX Experiences

Beyond production code, I design learning experiences that make complex AI and ML concepts tangible. These screens are from an interactive educational dashboard built for learners — featuring wizard-guided flows, live code generation, and explainability audits.

View 1 — Exploration
Model Landscape Map
An interactive scatter chart mapping 11 algorithms by complexity vs. versatility, built with a de-collision coordinate solver to prevent label overlap. Learners browse and select models from a searchable algorithm index.
View 2 — Guided Learning
Jimmy's Coding Spellbook
A 5-step wizard-guided pipeline builder where Jimmy the AI tutor walks learners through assembling a scikit-learn pipeline. Code is generated live as choices are made, bridging concept to implementation.
View 3 — Deep Dive
Parameter & Explainability Audits
Side-by-side model cards with explainability ratings, tunable hyperparameter counts, and Jimmy's analogies to demystify each algorithm's decision boundary — designed to reduce cognitive load for non-expert learners.
